What it does

Sprint at a glance
Auto-discovers your active sprint, shows full ticket list with assignees and story points.
Sentry issues by view
Fetches issues per saved view with correct project filters — collapsible per team.
Alert inbox
Velocity drop, goal risk, SLA breach, untriaged spikes — fires only when actionable.
Privacy mode
One-click toggle hides sensitive data for screen sharing.
Local-only storage
Credentials stay in Chrome storage — no backend, no servers, no tracking.
Light / Dark / Browser
Follows your OS theme or override it in settings.

Installation

  1. Download the latest .zip from Releases
  2. Unzip the folder
  3. Open Chrome → chrome://extensions
  4. Enable Developer mode (top-right toggle)
  5. Click Load unpacked → select the unzipped folder
  6. Click the extension icon → Set up credentials
  7. Enter your Jira URL, email, API token, Sentry org + token, and Sentry view IDs
